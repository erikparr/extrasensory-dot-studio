INSTALLATION
============

1. Double-click "VEX Installer.pkg"
2. Follow the installation wizard
3. Choose which formats to install:
   - VST3 (for most DAWs)
   - Audio Unit (for Logic Pro, GarageBand)
4. Restart your DAW

PLUGIN LOCATIONS
================

After installation:
- VST3: /Library/Audio/Plug-Ins/VST3/VEX MIDI EXPRESSION.vst3
- AU:   /Library/Audio/Plug-Ins/Components/VEX MIDI EXPRESSION.component

SYSTEM REQUIREMENTS
===================

- macOS 11.0 (Big Sur) or later
- Universal Binary (Intel + Apple Silicon)
- VST3 or Audio Unit compatible DAW
