WHAT IS VEX?
============

VEX MIDI EXPRESSION is a creative MIDI effect plugin that transforms
static MIDI CC (Control Change) messages into organic, living musical
expression using real-time physics simulation.

Turn boring, linear knob movements into:
- Bouncing, elastic motion with natural momentum
- Smooth spring oscillations with configurable damping
- Curved transformations with warp effects
- Randomized modulation with jitter

FEATURES:
=========

✓ Physics-Based Transformation
  - Bouncy: Charge-accumulation physics with gravity and collision
  - Spring: Damped oscillator with natural settling
  - Warp: Exponential/logarithmic curve mapping
  - Jitter: Random value modulation

✓ Multiple Simultaneous Mappings
  - Transform up to 8+ MIDI CC numbers at once
  - Each mapping has independent parameters
  - Real-time visual feedback

✓ Advanced Physics Controls
  - Bounciness (energy retention)
  - Gravity (acceleration strength)
  - Charge multiplier (responsiveness)
  - Kick cooldown (bounce frequency)
  - Orientation (ground/ceiling bounce direction)

✓ Professional Quality
  - Sub-10ms latency
  - 60fps physics simulation
  - Universal Binary (Intel + Apple Silicon)
  - Sample-accurate MIDI timing

✓ Cross-Platform Support
  - Windows: VST3
  - Linux: VST3
  - macOS: VST3 + Audio Unit + Standalone

QUICK START:
============

1. See "Installation_Instructions.txt" for detailed setup
2. Install the plugin format for your DAW
3. Insert VEX as a MIDI effect on a track
4. Add a mapping for your MIDI controller's CC number
5. Choose "Bouncy" physics
6. Move your controller and watch it bounce!
